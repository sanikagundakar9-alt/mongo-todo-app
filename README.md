# 📝 Smart MongoDB Todo App

A modern, responsive Full-Stack Todo application built with **Node.js**, **Express**, and **MongoDB Atlas**. This project demonstrates how to connect a backend server to a cloud database for permanent data storage.

## ✨ Features

* **Cloud Storage:** Data is stored permanently on MongoDB Atlas.
* **Modern UI:** Features a clean, responsive design with CSS gradients and hover effects.
* **Security:** Uses environment variables (`.env`) to keep database credentials safe.
* **CRUD Operations:** Supports adding and deleting tasks in real-time.

## 🛠️ Technologies Used

* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas (Cloud)
* **Object Modeling:** Mongoose
* **Security:** Dotenv (Environment Variables)
* **Frontend:** HTML5, CSS3 (Modern UI)

## 🚀 Getting Started

### 1. Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

### 2. Installation

Clone the repository or download the source code, then run the following command in your terminal to install dependencies:

```bash
npm install

```

### 3. Environment Setup

Create a file named `.env` in the root directory and add your MongoDB connection string:

```env
MONGO_URI=mongodb+srv://<username>:<password>@cluster0.xxxx.mongodb.net/?appName=Cluster0

```

### 4. Running the Application

Start the server by running:

```bash
node server.js

```

The app will be live at `http://localhost:3000`.

## 📁 Project Structure

* `server.js` - Contains the main server logic, database connection, and routes.
* `.env` - Stores sensitive database credentials (not uploaded to GitHub).
* `.gitignore` - Tells Git which files/folders to ignore (like `node_modules`).
* `package.json` - Lists project dependencies and scripts.
